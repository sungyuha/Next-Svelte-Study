Next.js 란

- React 프레임워크
    Next.js는 React 개발환경이 내장 되어 있음
    주로 React 개발 앱을 더 편리하게 구축하도록 많은 기능을 제공
    React 개발자가 편리하게 개발하는것에 목표
- 서버 사이드 렌더링 / 페이지 사전 렌더링
- Next.js는 서버에 요청이 오면 이 싱글페이지에 동적으로 사전 렌더링을 거쳐 콘텐트를 포함한 초기 페이지를 보여줌
- Next.js는 page 폴더가 중요함
/*
개발환경 실행 npm run dev
배포 파일 생성 npm run build
서비스 시작 npm run start
*/

/*
환경변수 : 프로세스가 컴퓨터에서 동작하는 방식에 영향을 미치는, 동적인 값들의 모임
.env에 담아 놓음
*/

Svelte 란

- 새로운 접근 방식의 프론드엔드 프레임워크
*장점 : 더 적은 코드를 작성 할 수 있음
    - 높은 가독성 유지
    - 개발 시간 단축
    - 쉬운 리팩토링
    - 쉬운 디버깅
    - 더 작은 번들(SPA 최적화)
    - 낮은 러닝 커브

vercel란

- Next.js 에서 제공하는 배포 플랫폼으로 [빌드 + 배포 + 호스팅] 서비스를 제공
